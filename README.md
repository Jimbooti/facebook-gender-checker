## Installation

### From Chrome Web Store (Recommended):
1. Visit the Chrome Web Store page for this extension
2. Click "Add to Chrome"
3. Click "Add extension" in the popup
4. Done! Go to Facebook and visit any profile

See INSTALL.txt included with your download

## Use Cases

- Verify friend requests before accepting
- Check suspicious profiles messaging you
- Quick screening of new connections
- Dating safety - verify profiles match their pictures

## Privacy

- Works 100% locally in your browser
- No data sent to external servers (except name analysis via genderize.io)
- Your flagged profiles are stored only on your computer
